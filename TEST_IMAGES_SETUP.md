## 🚀 사용 방법

### **방법 1: 앱에서 바로 사용 (권장)**

1. **브라우저에서 앱 열기**
   ```
   http://localhost:3000
   ```

2. **테스트 도구 카드 열기**
   - 대시보드 상단에 **"🧪 테스트 도구"** 카드가 있습니다
   - 카드를 클릭하여 펼치기

3. **바로 사용 버튼 클릭**
   ```
   [단서 1: 왼쪽 소매 물감]  [바로 사용] [다운로드]
   ```
   - **[바로 사용]**: 즉시 미리보기 화면으로 이동
   - **[다운로드]**: 이미지를 다운로드 (모바일 테스트용)

4. **분석 진행**
   - 미리보기 화면에서 **[✅ 이 사진으로 분석하기]** 클릭
   - 결과 확인

### **방법 2: 모바일 테스트**

1. **랩톱에서 이미지 다운로드**
   - 테스트 도구에서 **[다운로드]** 버튼 클릭
   - 3개 이미지 모두 다운로드

2. **모바일로 이미지 전송**
   - AirDrop (iPhone/Mac)
   - 카카오톡으로 자신에게 전송
   - 이메일로 전송
   - USB 케이블로 직접 전송

3. **모바일에서 테스트**
   - 앱 접속: `http://172.30.1.8:3000`
   - 🖼️ **보라색 버튼** (갤러리) 탭
   - 다운로드한 이미지 선택
   - 분석 진행
