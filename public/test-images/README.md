# 테스트 이미지 폴더

이 폴더에 테스트용 단서 이미지를 저장하세요.

## 📸 필요한 파일

다음 이미지 파일을 이 폴더에 복사하세요:

```
public/test-images/
├── clue1.jpg    # 단서 1: 왼쪽 소매에 묻은 물감
├── clue2.jpg    # 단서 2: 오른손에 쥐어진 붓
└── clue3.jpg    # 단서 3: 터펜타인 유리병
```

## 💡 사용 방법

이미지를 복사한 후 앱의 **"테스트 이미지 다운로드"** 버튼으로 바로 테스트할 수 있습니다.

## 🎯 파일명 규칙

- `clue1.jpg` → CLUE_01 응답
- `clue2.jpg` → CLUE_02 응답
- `clue3.jpg` → CLUE_03 응답

파일 형식은 `.jpg`, `.png`, `.webp` 모두 가능합니다.
